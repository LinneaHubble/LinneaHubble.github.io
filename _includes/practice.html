Data Science

Data science is an approach to solving problems using data in an iterative fashion. An experienced data scientist identifies key business metrics and attaches these metrics to a solution. Then, by using machine learning methods that target the identified metrics, the data scientist can optimize business goals directly.

In practice it's not always a direct path, but this is where experience paysoff. We have found a few ways of working with our customers.

  • One, the three day package where we work together with you to find an ideal approach to solve the current business challenge with the most accessible data. This gets processes focuses on measure the status quo of any current system and builds a proof of concept machine learning model to compare.
  • Two, hourly project based consulting is usually the case when a customer has a clear machine learning project in mind or already set up modelling processes.
  • Three, the proof of concept development or help.
  • Four, advisement on current process or laying the ground work for future data products.
